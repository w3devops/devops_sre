An 8-week journey to master cloud-native technologies and SRE principles

Welcome to My Learning Journey

This repository documents my comprehensive 8-week journey to master DevOps and SRE skills, preparing for a role involving Kubernetes, GitOps, observability, CI/CD, and cloud technologies. Each week focuses on specific technologies with hands-on projects, configurations, scripts, and troubleshooting logs.

Week 1: Kubernetes & Docker Foundation

Master Kubernetes/Docker basics, networking, security, and scripting.

  • Deployed Nginx to Minikube
  • Built Docker images for Flask apps
  • Configured Network Policy & RBAC
  • Image scanning with Trivy

Week 2: GitOps & Developer Experience

Master GitOps workflows, Kustomize, Helm, and developer collaboration.

  • Installed and configured ArgoCD
  • Created Kustomize manifests
  • Packaged apps as Helm charts
  • Developer guides and documentation

Week 3: Observability Basics

Learn monitoring and metrics with Prometheus and Grafana.

  • Installed Prometheus/Grafana
  • Created custom dashboards
  • Set up HPA and alerts
  • Debugged high-load issues

Week 4: Advanced Observability

Master logging, tracing, and secrets management.

  • Configured Loki for logs
  • Set up Tempo/Jaeger for traces
  • Instrumented with OpenTelemetry
  • Kubernetes Secrets management

Week 5: Service Mesh (Istio)

Learn Istio for microservices and security.

  • Installed and configured Istio
  • Deployed Bookinfo application
  • Configured routing and mTLS
  • Circuit breakers and traffic mirroring

Week 6: CI/CD & SRE Principles

Master CI/CD pipelines and SRE concepts.

  • Built Jenkins pipelines
  • Created Argo Workflows
  • Defined SLOs/SLIs
  • Error budget calculations

Week 7: Cloud & Infrastructure

Learn cloud-native Kubernetes and Infrastructure as Code.

  • Created EKS/GKE clusters
  • Used Terraform for provisioning
  • Cloud SQL integration
  • Cost analysis and optimization

Week 8: Advanced Topics

Databases, chaos engineering, and interview preparation.

  • Postgres with Operator
  • Chaos experiments with LitmusChaos
  • System design practice
  • Interview preparation
